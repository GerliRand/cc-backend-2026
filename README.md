# Sentiment Analysis App - Backend

This project is part of the **Cloud Computing course – Module 4 assignment**.

The backend provides a sentiment analysis API implemented in *Python using Flask*.
The API receives text input and returns a predicted sentiment generated by a trained machine learning model.

The backend is deployed on **CSC Rahti** and can be accessed over HTTPS.

## Technologies
- Python
- Flask
- Waitress
- Scikit-learn
- Docker

## Machine Learning Model
The sentiment prediction is generated using a trained machine learning model created earlier in **Module 2 assignment**.

The model was trained using:
* SSD review dataset
* TF-IDF vectorization
* LinearSVC classifier

The trained model is stored as: 
`sentiment_analysis_model.pkl`

The backend loads this model using Python **pickle** and uses it to generate predictions.

## Running the Backend Locally:
**1) Clone the repository:**
```
git clone https://github.com/GerliRand/cc-backend-2026 
```
**2) Navigate and open project folder in VS Code**
```
cd repository_name
open .
```
**3) Install dependencied (VS Code -> Terminal)**
```
pip install -r requirements.txt
```
**4) Start the development server:**
```
npm run dev
```
**5) Open in browser**
```
flask --app api run
OR
flask --app api run --debug
```

OR Using Waitress:
```
waitress-serve --host 127.0.0.1 api:app
OR
python app.py
```

## Running with Docker 
This project also includes Docker support.

Build and run container:
```
docker compose build
docker compose up
OR
docker compose up --build
```

The backend will be available at:
`http://localhost:5556`


## Deployment
The backend is deployed on **CSC Rahti** using a GitHub webhook pipeline.

Live backend endpoint: `https://cc-backend-2026-git-cloud-computing-rahti.2.rahtiapp.fi/ `
